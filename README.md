`https://jonathangetonapod.github.io/lgj-junior-strategist/`

## Local development

This repo has no build step. It is a single static page served from `index.html`.

Run a local server from the repo root:

```bash
python3 -m http.server 8000
```

Then open:
`http://127.0.0.1:8000/`

## Project structure

- `index.html` - page markup, styles, and inline JavaScript
- `logo.png` - LeadGenJay logo
- `jay-headshot.jpg` - founder image
